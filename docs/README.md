# public site 문서 운영

이 디렉터리는 2nd-brain public site 구현에 직접 관련된 운영 지식을 기록한다.

## 디렉터리

- `decisions/`: public site UI, Astro 렌더링, GitHub Pages 배포, 차트와 네비게이션 같은 구현 결정을 기록한다.
- `failures/`: public site에서 발생한 표시 오류, 빌드 오류, 레이아웃 문제, 데이터 렌더링 문제와 재발 방지 규칙을 기록한다.

## 기록 위치 판단

- public site 화면과 렌더링에만 영향을 주면 이 저장소의 `docs/`에 둔다.
- ingest, bot, DART, 위키 원천 데이터와 함께 봐야 하는 내용은 루트 저장소의 `docs/`에 둔다.
- 같은 내용을 두 저장소에 복사하지 않는다. 필요한 경우 링크로 연결한다.
