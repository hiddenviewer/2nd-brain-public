# 2nd Brain Public

Curated visual knowledge notes from a personal second brain, covering AI agents, Apple technologies, investing research, and future learning topics.

This repository is the public presentation layer for selected knowledge notes. It does not mirror the private vault directly. Notes are rewritten and structured for public reading, with source attribution and visual layouts where useful.

See [docs/content-pipeline.md](docs/content-pipeline.md) for the publishing pipeline.

## Scope

- Visual notes and explainers
- Category pages for AI, Apple, investing, language learning, travel, and future topics
- Searchable static site deployed with GitHub Pages
- Knowledge maps and timelines derived from curated public notes

## Not Included

- Private raw source files
- Inbox material
- Personal operating logs
- Unreviewed or private investment notes
- Full copies of third-party articles or documents

## Development

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
```
